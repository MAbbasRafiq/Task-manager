# 📝 Python Task Manager

A simple command-line task manager written in Python. It allows users to add, view, complete, and delete tasks, with data saved to a text file.

## 📂 Features

- Add new tasks
- View all tasks
- Mark tasks as complete
- Delete tasks
- Save/load tasks from a file (`tasks.txt`)
